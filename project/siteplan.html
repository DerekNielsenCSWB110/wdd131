Encoding & Ciphers - Site Plan

Site Name

Secret Code Academy
This name represents an educational site focused on cryptography, encoding techniques, and historical ciphers. It's memorable and clearly communicates the topic.

Optional domain availability: secretcodeacademy.com

Site Purpose

The site provides tutorials, examples, and interactive exercises about different encoding methods and ciphers. Users can learn about historical ciphers, modern encoding, and how to encrypt or decrypt messages safely.

Scenarios

  • How does the Caesar Cipher work and how can I decode it?
  • What is Base64 encoding and where is it used?
  • How can I encrypt a secret message safely online?

Color Schema

Primary color for headings: (Deep Blue) Secondary/accent color: (Orange)

Background color: #f4f4f4; used for page background. Paragraph text: #222; headings: #1a1a8c; accents/buttons: #d94f1c

Typography

  • Roboto Bold – headings (h1, h2)
  • Roboto Regular – body paragraphs and main content
  • Courier New – code blocks and examples for ciphers
